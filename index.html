React Data Grid

We think react rocks and use it a whole load at Adazzle. React Data Grid is a Excel-like grid component built with React, with editors, keyboard navigation, copy & paste, and the like. It's simple to get going with, but pretty powerful. And designed to be lightening fast even when used for large data sets with complex editing.
